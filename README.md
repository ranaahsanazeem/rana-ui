# Rana UI Framework
Built by Ahsan Azeem Zahid

## 📁 Project Structure
- `index.html`: Main entry point using React 18 & Babel CDN.
- `src/index.css`: Glassmorphic styling and professional UI themes.
- `src/App.jsx`: Component logic and AI Voice Tour integration.

## 🚀 How to Run
Kyunki hum CDN use kar rahe hain, aapko `npm install` ki zaroorat nahi. Bas `index.html` ko browser mein open karein.

## 🎤 Features
- AI Project Guide (Voice Assisted)
- Glassmorphic Cards & Precision Buttons
- Fully Responsive Layout
