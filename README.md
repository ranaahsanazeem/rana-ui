# 💎 Rana UI - Obsidian Framework

Rana UI ek professional-grade frontend project hai jo **React 18** aur **Glassmorphism** design aesthetics par mabni hai. Isme ek unique AI-powered voice guide feature shamil hai jo user ko poore project ka automated tour karwata hai.

## 📝 Project Description
Ye project ek "Obsidian" themed UI library hai jo modern web applications ke liye high-fidelity components provide karti hai. Isme subtle blurring, electric blue accents, aur smooth animations ka istemal kiya gaya hai taake ek premium tech feel mil sake.

## 📂 Project Structure
Aapka project niche diye gaye structure par mushtamil hai:
* **index.html**: Main entry point jo CDN ke zariye React aur Babel ko load karta hai.
* **src/index.css**: Saari professional styling aur glassmorphic effects isme maujood hain.
* **src/App.jsx**: Main logic aur UI components (Buttons, Cards, Modals) ka code yahan hai.
